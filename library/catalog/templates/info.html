О проекте

Этот сайт представляет собой каталог курсов, разработанный в рамках лабораторной работы по Django.

Особенности проекта:

  • Использование view-параметров в URL (/courses/1/, /authors/2/)
  • Хранение данных в константах (без использования БД)
  • Обработка ошибки 404 для несуществующих страниц
  • Навигация между всеми страницами сайта

Структура сайта:

  • Главная - общая информация
  • Курсы - список всех курсов
  • Авторы - список всех авторов

Все данные хранятся в файле catalog/data.py
